# 피크민 버섯 투력 계산기

피크민 블룸 버섯 탐험 투력을 간단히 계산하는 정적 웹 계산기입니다.

## 사용 방법

1. 피크민을 같은 조건끼리 묶어서 `마리`, `색상`, `밀접도`, `현재 꽃`, `데코 분류`를 입력합니다.
2. 위쪽의 `꽃을 준 뒤 상태`를 바꾸면 꽃을 먹였을 때의 예상 투력이 자동 계산됩니다.
3. 이벤트마다 달라지는 현재시즌/복각/일반 데코 보너스는 `데코 보너스` 영역에서 직접 수정합니다.
